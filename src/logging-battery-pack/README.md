# logging-battery-pack

A [battery pack](https://crates.io/crates/battery-pack) for logging and tracing in Rust.

## What's Included

| Crate | What it does |
|-------|-------------|
| [tracing](https://crates.io/crates/tracing) | Application-level tracing for async-aware structured logging |
| [tracing-subscriber](https://crates.io/crates/tracing-subscriber) | Utilities for composing tracing subscribers, with env-filter support |

## Quick Start

```sh
cargo bp add logging
```
